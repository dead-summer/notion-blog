7 运算符优先级

约 120 字小于 1 分钟

2024-12-01

L→R== !=
L→R&
L→R^
L→R`
L→R&&
L→R`
L→R? :
R→L= *= /= %=
+= -= <<= >>=
>>>= &= ^= `

只有单目运算符、赋值运算符是从右向左运算的。
