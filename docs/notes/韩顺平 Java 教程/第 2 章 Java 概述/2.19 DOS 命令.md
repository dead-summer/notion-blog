Dos: Disk Operating System, 即磁盘操作系统。常用的 dos 命令如下：

1. `dir` : 查看当前目录是有什么内容
2. `cd` : change directory. 如：

	切换到其他（如 D 盘）盘下：


	```bash
	cd /D d:
	```


	切换到当前盘的其他目录下：


	```bash
	cd <file-path>
	```


	切换到根目录：


	```bash
	cd \
	```

3. `tree` : 查看指定的目录下所有的子级目录
4. `cls` : 清屏
5. `exit` : 退出 DOS
