运算符有不同的优先级，所谓优先级就是表达式运算中的运算顺序。如下表，上一行运算符总优先于下一行。



| 结合顺序 | 运算符                            |
| ---- | ------------------------------ |
|      | `.`  `()`  `{}`  `;`  `,`      |
| R→L  | `++`  `--`  `~`  `!`           |
| L→R  | `*` `/` `%`                    |
| L→R  | `+` `-`                        |
| L→R  | `<<` `>>` `>>>`                |
| L→R  | `<` `>` `<=` `>=` `instanceof` |
| L→R  | `==` `!=`                      |
| L→R  | `&`                            |
| L→R  | `^`                            |
| L→R  | `|`                            |
| L→R  | `&&`                           |
| L→R  | `||`                           |
| L→R  | `? :`                          |
| R→L  | `=` `*=` `/=` `%=`             |
|      | `+=` `-=` `<<=` `>>=`          |
|      | `>>>=` `&=` `^=` `|=`          |


只有单目运算符、赋值运算符是从右向左运算的。
