对于整数，常见有四种表示方式：

1. 二进制：0,1 ，满 2 进 1.以 0b 或 0B 开头。
2. 十进制：0-9 ，满 10 进 1。
3. 八进制：0-7 ，满 8 进 1。以数字 0 开头表示。
4. 十六进制：0-9 及 A(10)-F(15)，满 16 进 1。以 0x 或 0X 开头表示。此处的 A-F 不区分大小写。
