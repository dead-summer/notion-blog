赋值运算符就是将某个运算后的值，赋给指定的变量。


赋值运算符包括：

- 基本赋值运算符：`=`
- 复合赋值运算符：`+=` , `-=` , `*=` , `/=`  , `%=`

赋值运算符有如下特点：

1. 运算顺序从右往左

	```java
	int num = a + b + c
	```

2. 赋值运算符的左边 只能是变量,右边 可以是变量、表达式、常量值

	```java
	int num = 20;
	int num2= 78 * 34 - 10;
	int num3 = a; // error
	```

3. 复合赋值运算符有如下等价效果。如：`a+=3` 等价于 `a=a+3` 其他类推。
4. 复合赋值运算符会进行类型转换，

	```java
	byte b = 2;
	b += 3; // 等价于 b = (byte)(b + 3)
	b++; // 等价于 b = (byte)(b + 1)
	```
